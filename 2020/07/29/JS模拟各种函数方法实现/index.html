JS模拟各种函数方法实现

2020-07-29 | technology
  • JS Call() 方法

    改变某个函数运行时的上下文 即改变函数体内部this的指向
    call(thisObj, arg1, arg2, arg3, arg4)
    thisObj 目标对象
    arg1,arg2,arg3,arg4 : 参数 对象

  • JS apply() 方法

    改变某个函数运行时的上下文 即改变函数体内部this的指向
    apply(thisObj, [args])
    thisObj 目标对象
    args：一个数组或类数组

  • JS bind() 方法

    创建一个函数 把函数体的this对象绑定到thisObj
    bind(thisObj, …args)
    thisObj 目标对象
    args：需要覆盖的传参
    配合 call和apply方法处理一下参数
    最后返回一个新函数 该函数的this指向目标对象

  • JS new 方法

    创建一个新的空对象
    把this绑定到空对象
    使空对象的proto 指向构造函数的原型(prototype# )
    执行构造函数，为空对象添加属性
    判断构造函数的返回值是否为对象，如果是对象，就使用构造函数的返回值，否则返回创建的对象

  • JS instanceof 方法

    该方法用来检测目标对象的原型链中是否存在某个构造函数的prototype属性
    遍历左边原型链 是否存在右边变量的 prototype
    存在即返回true 不存在返回false

  • JS 深拷贝

    判断类型是否为原始类型，如果是，无需拷贝，直接返回
    避免出现循环引用，拷贝对象时先判断存储空间中是否存在当前对象，如果有就直接返回
    开辟一个存储空间，来存储当前对象和拷贝对象的对应关系
    对引用类型递归拷贝直到属性为原始类型

  • JS 防抖函数

    函数防抖是在事件被触发n秒后再执行回调，如果在n秒内又被触发，则重新计时。使函数的多次调用合并为进行一次调用

  • JS 节流函数

    函数节流是在事件被触发n秒内，如果又被触发，则不触发事件。即在一定时间内只允许函数进行一次调用

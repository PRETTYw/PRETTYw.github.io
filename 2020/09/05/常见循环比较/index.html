常见循环比较

2020-09-05 | technology
  • 常见的各种循环主要作用于四种数据结构：Array Object Set Map
  • 数组和对象是基本数据类型 具体不介绍了
  • set 和 map 是es6新增的数据结构类型
  1. set拥有以下特点和方法、属性

    成员中的值是唯一的 不会重复
    添加成员时 不会发生类型转换
    添加的对象都不相等
    size：返回成员总数
    add 添加某个值，返回set本身
    has 返回一个布尔值，表示该值是否为set成员

  • 通常使用 Array.from(new Set(arr)) 去重
  • 使用arr=[…set]或者Array.from 把set类型的数据转成数组
  • set的遍历顺序就是插入顺序 set没有键名 只有键值 所有key方法和value方法返回的值一致
  1. map拥有以下特点和方法、属性
  • map是以键值对的形式存储数据 和对象差不多
  • size：返回map结构的成员总数
  • set(key,value)：设置键名key对应的键值value,然后返回整个map结构，如果key已经有值，则键值会被更新，否则就新生成该键
  • get(key)：读取key对应的键值，如果找不到key,则返回undefined
  • has(key)：返回一个布尔值，表示某个键是否在当前map对象中
  • delete(key)：删除某个key,返回true,如果删除失败，返回false
  • 使用arr=[…set]或者Array.from 把map类型的数据 转成数组

常见循环整理

  1. for 循环一般用于数组循环 for(var I=0;i<arr.length; I++) {} for循环对象时 对象只能是属性命名有一定规律的对象
  2. for in 在数组和对象的循环中都适用：for(var I in arr){if(arr.hasOwnProperty(i)){console.log(arr[i])}} 循环方法会让对象本身或者原型链上的都会被循环出来 可以用hasOwnProperty 来过滤掉原型上的属性
  3. for of 在array object set map 中都可以使用
  4. for each 在array object set map 中都可以使用
  • for用来循环数组 循环中可以使用return、break等来中断循环
  • for each 对数组的每一个元素执行一次提供的函数（不能使用return、break等中断循环），不改变原数组，无返回值undefined。
  • for in 循环遍历的值都是数据结构的键值 键名： o 键值： obj[o] 比较适合循环对象
  • for of 循环获取一对键值对中的值 一个数据结构只有部署了 Symbol.iterator 属性, 才具有 iterator接口可以使用 for of循环

    具有Symbol.iterator 属性的数据结构：：
    数组 Array 、Map、Set、String、arguments对象

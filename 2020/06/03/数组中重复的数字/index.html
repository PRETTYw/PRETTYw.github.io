数组中重复的数字（简单）

2020-06-03 | technology
  • 找出数组中重复的数字。在一个长度为 n 的数组 nums 里的所有数字都在 0～n-1 的范围内。数组中某些数字是重复的，但不知道有几个数字重复了，也不知道每个数字重复了几次。请找出数组中任意一个重复的数字。

输入：[2, 3, 1, 0, 2, 5, 3]
输出：输出：2 或 3
2 <= n <= 100000

  • 利用ES6中的Set 对象存储的值总是唯一的 特性 对象有add方法 循环往set数组中添加值 每次重复添加 set的数组长度不会改变 利用这个来判断 最后返回新数组就好了
  • 使用哈希表 遍历数组 检查是否出现过 如果出现过 返回此数字 题目要求返回一个数字就好了 所以返回第一个不是true的数组就好了
  • 拿到输入值后先排序 排序完挨个比较就好了 有相同直接输出就好了

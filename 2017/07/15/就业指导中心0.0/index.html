就业指导中心 QAQ

2017-07-15 | emotion

我们大三有个就业指导课，是涂老师给我们上的，因为马上要找实习了呢，所以在找实习过程中也要比较注意，所以就和大家分享一下老师的上课内容和最近找实习过程中遇到的事情。

就业注意事项

树立正确的就业观和择业观，不仅有利于自己正确的求职择业，迈好人生道路上的重要一步，而且有利于自己在今后的工作岗位上施展才华，最大程度地实现自己的人生价值。（官方正经话，还是要说一下0.0）

就我自己而言，我现在学校在江门，以后找工作希望能在广州，然后等年纪大了呢，希望回合肥（本人是安徽合肥的0.0）那时候应该比较壮了，毕竟父母都在合肥，母亲大人身体也不是很好，不想离他们太远。

如果要就业的话首先要了解所在地的政策，江门的就了解江门的，广州就找下广州的，百度一下都是有的0.0 还有入基层、入伍、自主创业都有相关的优惠政策，大家想了解的话可以百度

接下来是高校毕业生参加就业见习的相关政策：

  1. 见习单位应当与毕业生按照平等自愿，协商一致的原则签订见习协议
  2. 见习期限一般为3-6个月，最长不超过12个月

高校毕业生的入户政策

  • 取消高校毕业生落户限制
  • 鼓励高校毕业生到扶贫开发重点县工作
  • 毕业生户口的相关优惠政策
    大家感兴趣可以具体去了解一下，毕竟户口这事比较重要，了解清楚的好0.0

然后分享一下我最近找的实习，应该以前都没有经验，我就写好简历后，在智联还有网上百度一搜一大堆的招聘网站上点进去搜前端实习生，看了一下具体公司要求后就投了简历，后来就收到好多面试电话，那时候我才有点方，毕竟大姑凉出嫁第一次，我就百度公司名，结果搜到好多介绍都是骗人的，培训机构，我当时有点惊讶，我觉得找工作能有什么骗人的，最多就是让你做累点活，工资福利少点嘛，然后我就问了几个跟我一样找实习的童鞋和师兄，跟他们聊完后才知道找实习也有骗子，他们本身是培训机构，你去面试后他就说你可以上午跟师傅学，下午做些外包，然后结束后还要给培训费，外包帮他们做了还要给他们培训费，而且做外包不好的地方就是人家外包公司都有自己的一套，有自己的一些固定模版一些固定类库，你进去后就要学他的，那你以后跳槽后发现跳槽的公司和你学的都不一样，以前学的都用不上，那岂不是很吃亏。

跟同学还有师兄交流后我总结出了几个东西：（纯个人观点）

  • 看上公司先百度一下是不是真的 http://www.gsxt.gov.cn/index.html
  • 招聘软件首推BOSS直聘、实习僧、拉勾
  • 对于个人发展、全方面培养来说不推荐外包公司

我也是就业小白，这些都是我这段时间经历的，以后再慢慢完善，时间还有很多呢，等以后有新的东西我再继续往下写~

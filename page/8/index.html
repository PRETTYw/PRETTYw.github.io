给定数组 nums = [1,1,2]

[0,0,1,1,1,2,2,3,3,4] => [0,1,2,3,4,2,2,3,3,4] return [0,1,2,3,4].length == 5
返回的长度 = 原数组的长度 - 重复元素的长度
